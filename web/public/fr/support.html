Synergy

Soutien

Vous avez des problèmes avec Synergy ? Vous êtes au bon endroit !

Faire des recherches

Avant de demander à quiconque de l'aide, faites une recherche, cela prendra peu de temps. Avant de nous soumettre un problème, assurez-vous que sa solution ne soit pas déjà fournie. Vous pouvez également lire les documentations pour savoir comment utiliser les fichiers de configuration, etc.

Posez votre question

Si vous rencontrez des problèmes pour configurer ou pour démarrer avec Synergy, posez votre question sur le Synergy Q&A site. Lisez la FAQ pour vous assurer que votre question ne sera pas rejetée.

Liste de diffusion et de l'IRC

Les habitués des logiciels OpenSource peuvent préférer utiliser la liste de diffusion (cliquez simplement sur "Apply for group membership" - tout le monde est autorisé, c'est juste pour nous protéger contre les spammeurs). Mais, s'il vous plaît, ne postez pas les bugs sur la liste de diffusion. Vous pouvez aussi joindre notre canal IRC : #synergy on irc.freenode.net:8001 (pour IRC, vous pouvez utiliser XChat) - mais la même chose s'applique pour les IRC : faites une recherche avant !

Super User

Pour les habitués de Super User, il y a une étiquette "synergy".

  • Super User sert de site d'échange entre utilisateurs de Synergy.
  • Il n'est pas consacré au support de Synergy : les développeurs n'y sont pas actifs.
  • N'y soumettez pas de nouveaux bugs ; pour cela, utilisez le système de suivie de bug.
  • N'y postez pas vos propositions d'évolution du logiciel.
  • Utilisez l'étiquette "synergy" pour vos questions.
  • Assurez-vous de mentionner "Synergy" dans vos questions.

Rapporter un bug

Si vous avez épuisé toutes les autres options (ou si vous pensez que votre problème est vraiment un bug), vous pouvez ouvrir un nouveau compte et créer un nouveau rapport de bug.
