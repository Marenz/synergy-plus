Synergy

Soutien

Ayant des problèmes avec Synergy? Vous êtes au bon endroit!

Faire des recherches

Avant de demander à quiconque de l'aide, faire une recherche, il faudra un peu de temps. Vous devez bien étudier votre problème avant de nous demander de l'aide. Vous pouvez également lire les docs pour savoir comment utiliser les fichiers de configuration, etc.

Posez votre question

Si vous rencontrez des problèmes de configuration ou de départ Synergy, s'il vous plaît poser une question sur le Synergy Q&A site. Lisez la FAQ pour vous assurer que votre question ne sont pas fermées.

Liste de diffusion et de l'IRC

Plus traditionnelle utilisateurs open source peut préférer utiliser la liste de diffusion (cliquez simplement sur "Apply for group membership" - tout le monde est autorisé, c'est juste pour nous protéger contre les spammeurs). S'il vous plaît ne pas poster les bugs sur la liste de diffusion. Vous pouvez aussi joindre notre channel IRC: #synergy on irc.freenode.net:8001 (pour IRC, vous pouvez utiliser XChat) - mais la même chose s'applique pour les IRC; faire votre première recherche!

Super User

Si vous êtes familier avec le Super User, il ya un tag synergy.

  • Super User n'est pas consacré au soutien Synergy.
  • Posez des questions, ne pas harceler ou des commentaires sur les bogues ou problèmes.
  • Ne pas élever de nouveaux bogues là; utiliser le tracker problème pour cela.
  • Synergie développeurs ne pas de surveiller ce site.
  • Toujours utiliser la "synergy" tag sur votre question.
  • Assurez-vous que vous mentionnez "Synergy" dans la question.

Rapporter un bug

Si vous avez épuisé toutes les autres options (ou si vous pensez que votre problème est vraiment un bug), vous pouvez enregistrer un nouveau compte et puis ouvrez un nouveau rapport de bogue sur notre traqueur d'incidents.
