Synergy

Unterstützung

Haben Sie Probleme mit Synergy? Sie sind an der richtigen Stelle!

Betreibe Forschung

Bevor Sie jemanden um Hilfe, eine schnelle Suche, wird es eine kurze Zeit dauern. Sie muss voll Forschung das Problem, bevor fragen uns um Hilfe. Sie können auch lesen Sie die Dokumentation um herauszufinden, wie man Konfigurationsdateien, etc. verwenden.

Stellen Sie Ihre Frage

Wenn Sie Probleme mit der Konfiguration oder dem Beginn Synergy sind, bitte Stellen Sie eine Frage auf der Synergy Q&A Seite. Lesen Sie die FAQ, um sicherzustellen, dass Ihre Frage nicht geschlossen zu werden.

Mailing-Liste und IRC

Weitere traditionelle Open-Source-Benutzer können zu bevorzugen Nutzung der Mailingliste (klicken Sie einfach auf "Apply for group membership" - jeder ist erlaubt, dies ist nur, um uns vor Spammern zu schützen). Bitte senden Sie keine Fehler auf der Mailing-Liste. Sie können auch an unseren IRC-Channel: #synergy auf irc.freenode.net: 8001 (für IRC, können Sie Verwendung XChat) - aber das gleiche gilt für IRC, tun Sie Ihre Forschung zuerst!

Super User

Wenn Sie sich mit Super User sind, gibt es eine synergy tag.

  • Super User nicht für die Unterstützung Synergy gewidmet.
  • Fragen, nicht nörgeln oder zu kommentieren Fehler oder Probleme.
  • Nicht werfen neue Bugs gibt, die issue tracker dafür.
  • Synergy-Entwickler tun nicht Monitor dieser Website.
  • Verwenden Sie immer die "synergy"-Tag auf Ihre Frage.
  • Stellen Sie sicher, Sie erwähnen, "Synergy" in Frage.

Fehler melden

Wenn Sie schon alle anderen Optionen erschöpft (oder wenn Sie glauben, Ihr Problem ist wirklich ein Fehler), können Sie registrieren ein neues Konto und dann Öffnen einer neue Bug-Report auf unserer issue tracker.
