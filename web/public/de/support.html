Synergy

Support

Haben Sie Probleme mit Synergy? Sie sind an der richtigen Stelle!

Betreibe Forschung

Bevor Sie jemanden um Hilfe bitten, eine schnelle Suche, dauert nicht lange. Sie müssen erst gründlich selber Recherchieren, bevor fragen uns um Hilfe. Sie können auch die Dokumentation lesen um herauszufinden, wie man Konfigurationsdateien, etc. verwendet.

Stellen Sie Ihre Frage

Wenn Sie Probleme mit der Konfiguration oder dem Starten von Synergy haben, stellen Sie bitte eine Frage auf der Synergy Q&A Seite. Lesen Sie die FAQ, um sicherzustellen, dass Ihre Frage nicht geschlossen wird.

Mailing-Liste und IRC

Mehr traditionell veranlagte Open-Source-Benutzer bevorzugen eventuell die Nutzung der Mailingliste (klicken Sie einfach auf "Apply for group membership" - jeder ist erlaubt, dies ist nur um uns vor Spammern zu schützen). Bitte senden Sie keine Fehler an die Mailing-Liste. Sie können auch unserem IRC-Channel beitreten: #synergy auf irc.freenode.net: 8001 (für IRC, können Sie XChat benutzen) - aber das gleiche gilt für IRC, Forschung Sie zuerst selber nach dem Fehler!

Super User

Wenn Sie mit Super User vertraut sind, können Sie das synergy tag. benutzen

  • Super User ist nicht für die Unterstützung von Synergy verwandtwörtlich.
  • Stellen Sie Fragen! Nörgeln Sie an und kommentieren Sie Fehler oder Probleme nicht.
  • Tragen Sie keine neue Bugs dort ein, dafür ist der issue tracker!.
  • Synergy-Entwickler verfolgen die Webseite nicht aktiv.
  • Verwenden Sie immer das "synergy" Tag bei Ihren Fragen.
  • Stellen Sie sicher "Synergy" in der Frage zu erwähnen.

Fehler melden

Wenn Sie schon alle anderen Optionen erschöpft haben (oder wenn Sie glauben, Ihr Problem ist wirklich ein Fehler), können Sie ein neues Konto registrieren und dann ein neuen neue Bug-Report auf unserer issue tracker eröffnen.
